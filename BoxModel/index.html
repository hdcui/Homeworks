左图为两种不同的盒子模型渲染方式，在设定相同属性条件下的渲染效果对比。左图一为标准盒子模型(content-box)渲染效果，左图二为怪异盒子模型(border-box)渲染效果。从图中可以明显的看出两种渲染方式不同，标准盒模型content部分保持不变，所有设定的padding及border向外延伸；怪异盒子模型盒子大小保持不变，所有设定的padding及border向内部挤压，因此两种渲染方式会造成盒子content区域的巨大差异。请参考代码差异，差异部分用红色字体highlight：

box-sizing: content-box;

width: 200px;
height: 200px;
border: 20px solid rgb(0, 0, 0);
margin: 50px;
padding: 50px;
background-color: rgb(0, 255, 0);
float: left;
}
